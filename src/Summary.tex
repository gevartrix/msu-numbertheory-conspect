{\Large Асимптотический закон распределения простых чисел}

\begin{itemize}
	\item Вводим функции $\pi(x), \theta(x), \psi(x)$, и доказываем связь между ними.
    \item Лирическое отступление в виде оценок Чебышева.
    \item Введение функции $\zeta(s)$.
    \item С воспоминаниями про арифметические функции нашли $\frac{\zeta'(s)}{\zeta(s)}$.
    \item Доказательство того, что $\zeta(1 + it) \ne 0$
    \item Построение вспомогательной $F(s) = -\frac{1}{s} \frac{\zeta'(s)}{\zeta(s)} - \frac{1}{s - 1}$
    \item Доказательство того, что $F(1) < \infty$
    \item $\displaystyle F(1) =  \int_{1}^{\infty} \frac{\Psi(x) - x}{x^2} < \infty \Rightarrow \Psi(x) \sim x$, откуда и следует АЗРПЧ.
\end{itemize}
